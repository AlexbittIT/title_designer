# Титры в эфир (vMix)

Локальная система титров: вы задаёте текст и графику в панели управления, а в vMix добавляете один источник **Web Browser** с URL вывода — титры обновляются в реальном времени.

## Быстрый старт

1. **Запуск сервера** (в папке проекта):
   ```bash
   node server.js
   ```
   Или дважды кликнуть по `start.bat` (если создан).

2. Откроются два адреса:
   - **Панель управления:** http://127.0.0.1:7355/
   - **Вывод для vMix:** http://127.0.0.1:7355/output

3. **В vMix:**
   - **Add Input** → **Web Browser**
   - **URL:** `http://127.0.0.1:7355/output`
   - **Width / Height:** под ваш проект (например 1920 x 1080)
   - Добавить источник на сцену и выводить в эфир при необходимости.

4. В браузере на http://127.0.0.1:7355/ вводите текст, URL картинки, положение, нажимаете **Сохранить** и при необходимости **Показать** / **Скрыть** титры. Страница `/output` (и vMix) обновляются автоматически.

## Что можно настроить

- **Три строки текста** — заголовок, подзаголовок, доп. строка.
- **Графика** — URL картинки (логотип, баннер). Можно использовать файлы из папки `public/`, например `/public/logo.png`.
- **Положение** — внизу, вверху или по центру экрана.
- **Показать / Скрыть** — титры остаются в памяти, но на выводе можно их временно скрыть.

## Папка `public/`

Положите сюда свои картинки (логотипы, фоны). В панели в поле «URL картинки» укажите, например: `/public/logo.png`.

## Требования

- Установленный **Node.js** (например с https://nodejs.org/).
- vMix с поддержкой источника **Web Browser** (встроенный браузер на Chromium).

## Порт

По умолчанию используется порт **7355**. Если нужно изменить — отредактируйте константу `PORT` в начале файла `server.js`.
